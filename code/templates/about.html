FLOOD PREDICTION SYSTEM

Machine Learning Based Early Flood Warning System

Project Overview

Floods are among the most devastating natural disasters, causing significant damage to life, property, agriculture, and infrastructure. This project uses Machine Learning techniques to predict the possibility of floods based on weather and environmental parameters. The application helps authorities and disaster management teams take preventive actions by providing early flood predictions.


Project Objectives

  • Predict flood occurrence using Machine Learning.
  • Provide an early warning system.
  • Support disaster management authorities.
  • Reduce loss of life and property.
  • Improve disaster preparedness.

Machine Learning Algorithms

Algorithm Purpose
Decision Tree Classification
Random Forest Improved Classification Accuracy
K-Nearest Neighbors Similarity Based Prediction
XGBoost Best Performing Prediction Model

Technologies Used

  • Python
  • Flask
  • Pandas
  • NumPy
  • Scikit-Learn
  • XGBoost
  • HTML5
  • CSS3
  • Bootstrap

Input Parameters

  • Annual Rainfall
  • Seasonal Rainfall
  • Cloud Visibility
  • Humidity
  • Temperature
  • River Discharge
  • Soil Moisture

Expected Output

The system predicts whether there is a possibility of flooding based on the given weather conditions. The prediction result helps users make informed decisions and improve disaster preparedness.
