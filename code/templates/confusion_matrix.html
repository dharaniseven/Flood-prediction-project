What is a Confusion Matrix?

A Confusion Matrix is used to evaluate the performance of a Machine Learning classification model. It compares the predicted values with the actual values and shows how many predictions are correct and incorrect.


Term Description
True Positive (TP) Flood correctly predicted.
True Negative (TN) No Flood correctly predicted.
False Positive (FP) No Flood predicted as Flood.
False Negative (FN) Flood predicted as No Flood.
